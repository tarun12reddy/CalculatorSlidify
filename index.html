Introduction

  1. Aim of the Project is do show a demo of calculator using R Shiny

  2. In ui.R we give inputs and in server.R we use these inputs to estimate the outputs

Examples

  1. Addition:

    a = 1
b = 2
c = a + b
print (c)

    ## [1] 3

  2. Subtraction

    a = 1
b = 2
c = a - b
print (c)

    ## [1] -1

  1. Multiplication

    a = 1
b = 2
c = a * b
print (c)

    ## [1] 2

  2. Division

    a = 1
b = 2
c = a / b
print (c)

    ## [1] 0.5
      Note
    • b cannot be zero
